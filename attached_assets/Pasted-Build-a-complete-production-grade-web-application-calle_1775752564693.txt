Build a complete, production-grade web application called "SwasthyaSaar AI" 
— an AI-powered intelligent care orchestration system for India's frontline 
health workers (ASHA workers). This is for a Harvard HSIL Hackathon 2026 
and must look world-class, professional, and demo-ready.

====================
TECH STACK
====================
- React + Vite (frontend)
- Tailwind CSS (styling)
- React Router v6 (navigation)
- Recharts (charts/analytics)
- Web Speech API (voice input)
- Lucide React (icons)
- Mock data only — no real backend needed, use useState/localStorage
- Mobile-first, fully responsive

RISK SCORING LOGIC (Frontend Mock)
====================

MATERNAL RISK SCORE (calculate in JavaScript):
- Age < 18 or > 35: +20 points
- Hemoglobin < 7: +25 points, < 10: +15 points
- BP Systolic > 140: +30 points
- ANC visits < 3: +20 points
- Any danger sign checked: +10 each
- Previous complications: +15 points
- Distance > 10km: +10 points
- Total 0-40: LOW, 41-60: MEDIUM, 61-80: HIGH, 81+: CRITICAL

NCD RISK SCORE:
- Fasting sugar > 126: +30, > 100: +15
- BP > 140: +25
- Age > 45: +15
- Smoking: +20
- Family history: +10 each
- Sedentary lifestyle: +15
- Total scoring similar bands

NEWBORN RISK SCORE:
- Temperature > 38 or < 36: +25
- Not feeding: +30
- Abnormal breathing: +30
- Blue/pale color: +35
- Weight < 2kg: +20
